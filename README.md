## CRUD App with React Redux and JSON-Server

### 🔴 How to run Backend?
Go to **server** folder and run the command
```
npm start
```
➡️Backend **http://localhost:3009/users**

### 🔴 How to run Frontend?
Go to **crud-f** folder and run the command
```
npm start
```

➡️Frontend **http://localhost:3000**
